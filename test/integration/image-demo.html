Brix Imageserver Integration demo (with PAF)

This test will require to have the AMS server (Java webapp in Tomcat) and Brix test Servers (IPS & CorrectnessEngine) up with the IPS pointing to a local deployment of the Brix image repo.

Remember to configure laspaf.properties to NOT use the PAF Stub. This page is currently pointing at the PAF Dev env.
